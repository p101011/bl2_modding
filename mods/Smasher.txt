#<Smasher>

    #<Description>

        By Aaron0000

        Features;

        Replaces the lousy starter pistol you get from Claptrap with the Pre-Sequel's Smasher.

        Has two options for the stats, the original stats or the re-balanced version (More damage, two ammo per shot)

        Note;

        Will absolutely delete any normal Dahl Starter pistols you may have lying around. Smashers will also be deleted if you try to run a save without this mod active.

        It uses the Master material for Jakobs Legendaries in order to give it a skin, so anything that uses it as a parent will be affected (only the Hammer Buster in normal cases, but that is fixed at the bottom of this file.)

        Gibbed Code for a Level 1 Smasher; BL2(hwAAAAD1fwCHxwJABhEIQMOIhdENIwHG/////x0w/v97wwCD8Q3j)
