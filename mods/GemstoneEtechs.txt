#<Gemstone E-Tech Weapons>

    #<Description>

        By Aaron0000

        Enables Gemstone variants of E-Techs to spawn.

        NOTES;

        The game will delete the weapons if you do not run the mod before loading a save file.

        These E-Techs operate in the same way Hyperion Diamond E-Tech Pistols do, meaning they can spawn with no element, or accessories that otherwise wouldn't be present on the normal, non-Gemstone version.

        Due to hotfix "brilliance", Aquamarine Railers won't work unless you do the following;

        1. Load up a file that doesn't have the Aquamarine Railer, and start up a game.

        2. Activate this mod.

        3. Exit back to main menu.

        4. Activate this mod.

        5. Load the file with the Aquamarine Railer on it.

    #</Description>
