#<Revolvers>

    #<Description>

        By Aaron0000

        Features:

        Changes the Jakobs pistol barrel to behave like the Revolvers of Borderlands 1. Includes options for Maggie, Greed, Rex, Unforgiven, and Unofficial Community Patch Judge (Off by default).

        High damage.

        High accuracy.

        +100% critical hit damage.

        Lowered fire rate.

        Lowered reload speed.

        Recoil is larger.

        Consumes 3 ammo per shot (Maliwan consumes 4 per shot.)

        Typically six shots, can be higher or lower depending on manufacturer/grip/accessory/rarity.

        NOTES:

        If using the Luck Cannon, there is a fix for it at the bottom.

        If using S&S Forever, these is a fix for it at the bottom.

        If using the Defiler (my version), there is a fix for it at the bottom.

        If using Atlas Reborn's Aries, there is a fix for it at the bottom.

        If using the Cyber Eagle, there is a fix for it at the bottom.

        If using the Triple from The Campin' Dutchpack, there are fixes for it at the bottom.
